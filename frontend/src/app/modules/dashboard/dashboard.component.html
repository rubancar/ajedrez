CHESS MANAGER

Programación del lado del cliente y visualización (TWCAM)


Con este trabajo se busca realizar una pagina web implementada en angular que permita la gestion de los diferentes elementos de una federacción de Ajedrez.

En esta aplicacion se pueden crear, editar, listar, visualizar y eliminar:

  • Jugadores.
  • Partidas.
  • Torneos.
  • Clubes.
  • Entrenadores.
  • Federaciones.

Tambien cuenta con acciones especiales como:

  • Establecer resultados de partida.
  • Gestionar el calendario de calentamiento de un entrenador.
